# Mono Layout Guide

`mono-layout` コンポーネントは、要素の配置（レイアウト）を簡単に制御するための機能です。
主に横並びの `@[row]` と、縦並びの `@[stack]` を提供しています。

## 1. 基本的な使い方

### 横並び (`@[row]`)

要素を横に並べる場合に使用します。PC環境では横並びに、モバイル（画面幅が狭い場合）では自動的に縦並びにフォールバックします。

```markdown
@[row]
:::column
1つ目の要素
:::
:::column
2つ目の要素
:::
@[/row]
```

### 縦並び (`@[stack]`)

要素を強制的に縦に並べる場合に使用します。

```markdown
@[stack]
:::column
上の要素
:::
:::column
下の要素
:::
@[/stack]
```

## 2. 余白 (Gap) の設定

デフォルトでは、カラム間の余白はありません（ピタッとくっつく状態です）。
要素間を少し離したい場合は、引数として `class` 属性を指定します。

用意されている余白用のクラスは以下の3種類です。

- `gap-sm`: 小さな余白 (0.5rem)
- `gap-md`: 中くらいの余白 (1.5rem)
- `gap-lg`: 大きな余白 (3.0rem)

**使用例:**

```markdown
@[row](class="gap-md")
:::column
要素A
:::
:::column
要素B
:::
@[/row]
```

## 3. 配置 (Alignment) の設定

要素の配置や揃え位置を変更するためのクラスも用意されています。

- `center`: 中央揃え（上下左右の中央）
- `start`: 上部・左側寄せ
- `end`: 下部・右側寄せ

複数のクラスを同時に指定することも可能です（スペース区切りで指定します）。

**使用例（中央揃えで余白も指定する場合）:**

```markdown
@[row](class="gap-md center")
:::column
要素A
:::
:::column
要素B
:::
@[/row]
```

## 4. フォントサイズの設定について

現状、レイアウトコンポーネント自体の引数でフォントサイズを一括指定する機能はありません。
文字の大きさを変更したい場合は、Markdown内で HTML タグを使用し、直接インラインスタイルを記述してください。

**使用例:**

```markdown
@[row](class="gap-lg")
:::column
<p style="font-size: 0.75rem;">
  Z：通常ページの視線移動。左上から右方向に移動し、中身をざっと確認しながら、最終的に右下へ辿り着く。
</p>
:::
:::column
<p style="font-size: 0.75rem;">
  F：リスト形式の視線移動。左側を下方向に移動し、興味があれば右方向へ移動する。
</p>
:::
:::column
<p style="font-size: 0.75rem;">
  N：縦書き（日本語）ページの視線移動。
</p>
:::
@[/row]
```
